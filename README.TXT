WARNING: this version of the *installer* integrates thirdparty JWplayer
as an optional playback option (installed in "thirdparty" folder).
JWPlayer is under a Creative Commons Non-Commercial license.
For commercial use of JWplayer you must buy a license:
http://www.jeroenwijering.com/?item=JW_FLV_Media_Player

This version of the installer is then also under a Creative Commons license:
http://creativecommons.org/licenses/by-nc-sa/3.0/

For each recording, a folder is created in your home folder, for example
C:\documents and settings\yourlogin\audiovideocours\date-time\

The default playback file is cours.smil wich will be read by Real Player.
A "recording.html" file is also created and use JWplayer; for all sync
 features to work the recording folder must be copied on a server.
